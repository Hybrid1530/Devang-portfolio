IoT Hydroponics Monitor

A smart system built using STM32, DHT11, and MQTT protocol, integrated with Node-RED and ThingSpeak for visual monitoring and automation of plant care in hydroponics environments.

Technologies Used

  • STM32 Microcontroller
  • DHT11 Temperature and Humidity Sensor
  • MQTT Protocol
  • Node-RED Dashboard
  • ThingSpeak Integration
