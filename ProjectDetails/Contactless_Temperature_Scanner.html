Contactless Temperature Scanner

Built using MLX90614 IR temperature sensor, STM32 microcontroller, and ultrasonic sensor for distance-based contactless scanning. Used for public entry screening.

Technologies Used

  • MLX90614 IR Temperature Sensor
  • Ultrasonic Distance Sensor
  • STM32 Microcontroller
